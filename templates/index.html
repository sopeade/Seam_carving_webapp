Seam Carving App

❮ ❯

This is my implementation of the Computer Vision Technique "Seam Carving", based on the paper: Content-Aware Image Resizing, for resizing an image based on its contents. Regular image modification leaves unwanted distortions. E.g. Cropping cuts off peripheral content from an image whereas adjusting the image width or height distorts the contents by stretching or compressing.

Seam Carving however works by removing pixels with the "lowest cumulative enery" from within the image. Linear connected pixels of the image ("Seams") where the image changes the least are calculated mathematically (via minimizing the derivative of the pixel values of the image), and removed. The effect of removing multiple seams is a resized image with minimal effect on internal content. Note: Large seam removal percentages will likely still incur image artifacts (i.e. mishapen edges, pixel misalignment etc.) as such Seam removal has been capped at 40%. Select an image file to "seam carve" your own image!
