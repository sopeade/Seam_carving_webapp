Seam Carving App

❮ ❯

This app is called Seam carving. This is my implementation of the computer Vision technique for adjusting an images size based on its contents (content aware resizing). It is based on the paper: Content-Aware Image Resizing. Regular image modification (e.g. via cropping), works by cutting off content from the boundary of the image. Cropping is a blunt tool functioning regardless of the content of the image. Adjusting the image width or height distorts the contents. This technique on the other hand works by removing pixels from within the image ("seams") based on a concept of "least cumulative enery". Linear connected pixels of the image where the image changes the least(a seam) are calculated mathematically (via minimizing the derivative of the pixel values of the image), and removed. The cumulative effect of removing multiple seams results in an image which can be cropped with minimal effect on internal content. Seam removal is currently fixed to 30%, empirically determined to minimize image artifacts (i.e. mishapen edges, pixel misalignment etc.) Select an image file to "seam carve" your own image!
